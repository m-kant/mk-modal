$.mkmodal Demo

mkmodal is a jQuery plugin for modal dialog window, that uses CSS animation. Dialog content can be taken from string, DOM element or file via ajax. 

$.mkmodal.open(options); // general form
$.mkmodal.acquire(".dialog-content"); // take content from DOM Element
$.mkmodal.load("dialogs/dialog.html"); // load content via AJAX
